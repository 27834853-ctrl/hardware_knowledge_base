- 欧姆定律：V = I × R

- 容抗公式：Xc = 1/(2πfC)
- 感抗公式：XL = 2πfL
- RC时间常数：τ = R × C
- LC谐振频率：f = 1/(2π√LC)
- 电容充电：V(t) = V₀(1 - e^(-t/τ))
- 电感电压：V = L × dI/dt
- 电感能量：E = ½LI²
- 电容能量：E = ½CV²
- 分压公式：Vout = Vin × R2/(R1+R2)
- 基尔霍夫电流定律：ΣI = 0
- 基尔霍夫电压定律：ΣV = 0

#### 中级篇（25个）
- 反射系数：Γ = (ZL - Z0)/(ZL + Z0)
- 反射电压：Vr = Γ × Vi
- 串扰系数：Kc = C_m/C_g
- 地弹电压：ΔV = L × dI/dt
- 微带线阻抗：Z0 = 60/√εᵣᵉᶠᶠ × ln(8h/w + w/(4h))
- 带状线阻抗：Z0 = 60/√εᵣ × ln(4b/(0.67πw(0.8+w/h)))
- 有效介电常数：εᵣᵉᶠᶠ = (εᵣ+1)/2 + (εᵣ-1)/2 × 1/√(1+12h/w)
- 差分阻抗：Zdiff = 2 × Z0 × (1 - k)
- 耦合系数：k = e^(-πs/h)
- LDO功耗：Pd = (Vin - Vout) × Iout
- 热阻计算：Tj = Ta + θja × Pd
- Buck电感：L = (Vin-Vout) × Vout / (Vin × ΔI × fsw)
- Buck输出纹波：ΔV = ΔI / (8 × f × C)
- I2C上拉电阻：Rp = (Vcc - Vol) / (Iol × n)
- 传输线延迟：td = √(εᵣᵉᶠᶠ) / c
