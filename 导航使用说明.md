## 📍 如何访问高速接口深度内容

### 方式 1: 使用左侧目录导航（推荐）

1. **打开页面**: https://27834853-ctrl.github.io/hardware_knowledge_base/
2. **找到左侧目录**: "高级篇" → "高速接口深度技术"
3. **点击展开**: 鼠标悬停或点击即可展开子菜单
4. **选择接口**: 点击任意高速接口名称

**6 大高速接口选项**:
- 🔬 **PCIe Gen 1-6 深度测试**
  - TX/RX 测试、LTSSM 11 状态、均衡技术、PAM4 编码

- 🔌 **USB 3.2/4.0 深度测试**
  - USB4 Tunneling、PD 3.1 EPR 240W、Type-C Alt Mode

- 📷 **MIPI D-PHY/C-PHY 测试**
  - 三相编码、16-to-7 映射、CSI-2 v4.0、DSI-2 v2.0

- 💾 **LPDDR5X 8533 MT/s 训练**
  - CA Training、Write Leveling、Read Training、WCK2CK

- 🖥️ **DisplayPort 2.1 UHBR**
  - UHBR10/13.5/20、Adaptive Sync (VRR)、DSC 1.2a、TX/RX 均衡

- 📺 **HDMI 2.1a FRL + eARC**
  - FRL 48 Gbps、eARC 音频回传、VRR/ALLM/QMS 游戏特性

### 方式 2: 直接点击内容区卡片

2. **查看 6 个导航卡片**: 每个卡片代表一个高速接口

4. **HDMI VRR/ALLM/QMS** → 游戏优化技术

## 🆘 常见问题

### Q1: 点击目录后没有反应？

**解决方案**:
1. 检查是否点击了正确的"高速接口深度技术"分组
2. 确保 JavaScript 已启用
3. 刷新页面（F5）重新加载
4. 清除浏览器缓存

### Q2: 内容加载缓慢？

**解决方案**:
- 第一次加载时，high-speed-deep-dive.js (7,050 行) 需要时间
- 建议使用 Chrome、Edge、Firefox 最新版本
- 本地预览：使用 `python -m http.server 8080`

### Q3: 公式显示为原始代码？

**解决方案**:
- 等待 MathJax 库加载完成（约 2-3 秒）
- 刷新页面
- 检查网络连接（MathJax 从 CDN 加载）

### Q4: 卡片点击后内容在哪？

**位置**: 卡片下方的"内容显示区域"
- 如果看不到，向下滚动页面
- 内容区有淡入动画（0.5秒）

### Q5: 如何打印或导出内容？

**方法**:
1. 点击要导出的接口卡片，内容展开
2. 浏览器 → 打印 → 另存为 PDF
3. 或使用浏览器阅读器模式
