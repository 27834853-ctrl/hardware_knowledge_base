## 🔧 启动方式详解

### 方式 1: Python HTTP Server（推荐）

```bash
cd "D:\App_Collection\Microsoft VS Code\hardware_knowledge_base"
python -m http.server 8080
```

**优点**:
- Python 自带，无需安装
- 简单可靠
- 适合本地测试

**访问端口**: 8080
**停止服务**: `Ctrl+C`

---

### 方式 2: VS Code Live Server

1. 在 VS Code 中打开项目文件夹
2. 右键点击 `index.html`（或其他 HTML 文件）
3. 选择 "Open with Live Server"

**优点**:
- 自动刷新
- 集成在 VS Code 中
- 方便开发调试

**默认端口**: 5500

---

### 方式 3: Node.js http-server

```bash
# 全局安装（仅需一次）
npm install -g http-server

# 启动服务器
cd "D:\App_Collection\Microsoft VS Code\hardware_knowledge_base"
http-server -p 8080
```

**优点**:
- 功能更强大
- 支持更多配置
- 适合专业开发

## 🐛 常见问题

### Q1: 启动服务器后无法访问？

**检查**:
```bash
# 确认服务器正在运行
# 应该看到类似输出：
# Serving HTTP on 0.0.0.0 port 8080 (http://0.0.0.0:8080/) ...
```

**解决**:
- 检查端口是否被占用：换成 8081、8082 等
- 检查防火墙设置
- 使用 `127.0.0.1:8080` 替代 `localhost:8080`

### Q2: test-simple.html 显示错误？

**可能原因**:
- high-speed-deep-dive.js 未加载
- 文件路径错误

**解决**:
- 确保从项目根目录启动服务器
- 检查浏览器控制台 (F12) 的错误信息

### Q3: 内容显示不完整？

**解决**:
- 清除浏览器缓存 (Ctrl+Shift+Delete)
- 强制刷新 (Ctrl+F5)
- 重启服务器

### Q4: Python 命令不存在？

**Windows 系统**:
```bash
# 尝试 python3
python3 -m http.server 8080

# 或使用 py
py -m http.server 8080
```

**没有 Python？**
- 下载安装: https://www.python.org/downloads/
- 或使用 VS Code Live Server
